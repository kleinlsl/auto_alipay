## auto.js蚂蚁森林收能量
* 在网上看了一些自动收能量的脚本
* 写了一个脚本
* 使用AutoJs运行
	* AutoJS下载地址 :[here](https://github.com/kleinlsl/auto_alipay/releases/download/auto_alipay_v1.0/Auto.js.apk)
* 运行步骤如下：
	* 1.解锁（点亮屏幕，切换到输入密码界面，输入密码）
	* 2.打开支付宝
	* 3.找到蚂蚁森林
	* 4.收取自己的能量
	* 5.偷取他人能量
